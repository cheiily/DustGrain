# dustgrain-core
This is the core package for dustgrain - a mediawiki [cargo](https://www.mediawiki.org/wiki/Extension:Cargo) data extraction tool, intended for use with [dustloop.com](https://www.dustloop.com/w/Main_Page).
It contains the bulk of fetching & processing logic. 

If you so desire, you can use this package directly, but it's recommended to instead use the dustgrain-jvm package as a library, or the dustgrain-cli project for a command-line interface.
